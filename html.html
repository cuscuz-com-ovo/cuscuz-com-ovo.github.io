 Elementos HTML 


 Dentro do html existem alguns elementos que são utilizados com o intuito de melhor organizar o código que será construido.


Os elementos são: html, body, h1 e p. Todos eles serão explicados de forma detalhada logo abaixo.


 Os elementos HTML podem ser aninhados (isso significa que os elementos podem conter outros elementos).
    
Elemento Html

    
Todos os documentos HTML consistem em elementos HTML aninhados.

    
O exemplo a seguir contém quatro elementos HTML ( , <html>e ) :<body><h1><p></p></p>
        
            
EXEMPLO:


<!DOCTYPE html>
    <html>
        <body>
    
            <h1>My First Heading</h1>
            <p>My first paragraph.</p>
    
            </body>
            </html>




  
    
Elemento Body
 

O <html>elemento é o elemento raiz e define todo o documento HTML.
 
    
Ele tem uma tag inicial <html>e uma tag final </html>.

    
Então, dentro do <html>elemento existe um <body> elemento:

    
        
EXEMPLO:


<body>

    <h1>My First Heading</h1>
    <p>My first paragraph.</p>
    
    </body>





    
Elemento H1 e P


O <body>elemento define o corpo do documento.

   
Ele tem uma tag inicial <body>e uma tag final </body>.
 
    
Então, dentro do <body>elemento existem dois outros elementos: <h1>e <p>:

    
        
EXEMPLO:


<h1>My First Heading</h1>
    <p>My first paragraph.</p>





    
Elemento H1


O <h1>elemento define um cabeçalho.

   
Ele tem uma tag inicial <h1>e uma tag final </h1>:

   
    
EXEMPLO:


<h1>My First Heading</h1>




    
Elemento P


O <p>elemento define um parágrafo.

    
Ele tem uma tag inicial <p>e uma tag final </p>:

    
        
EXEMPLO:

    
<p>My first paragraph.</p>
    

    

     Cabeçalho HTML
    
Cabeçalhos HTML são títulos ou subtítulos que você deseja exibir em uma página da web.

    
Os cabeçalhos HTML são definidos com as tags <h1>to <h6>.
        
<h1>define o cabeçalho mais importante. <h6>define o título menos importante.

    
        
EXEMPLO:

    
 
 <h1> Heading 1 </h1> 

       
<h2> Heading 2 </h2> 

        
<h3> Heading 3 </h3> 

        
<h4> Heading 4 </h4> 

        
<h5> Heading 5 </h5> 

        
<h6> Heading 6 </h6> 

    

    

    
Os navegadores adcionam um espaço em branco (uma margem) automaticamente como vocês podem ver na imagem acima.

        
Títulos

    
Os mecanismos de pesquisa usam os cabeçalhos para indexar a estrutura e o conteúdo de suas páginas da web.

        Os usuários geralmente percorrem uma página por seus títulos. É importante usar cabeçalhos para mostrar a estrutura do documento.
        
        <h1>os cabeçalhos devem ser usados ​para os cabeçalhos principais, seguidos pelos <h2>cabeçalhos, depois os menos importantes <h3>e assim por diante. 

        
É interessante que você use a opção de cabeçalho quando realmente for construir um cabeçalho, e não apenas para tornar textos grandes e com margens maiores.

        
Cabeçalhos grandes

        
Cada título HTML tem um tamanho padrão. No entanto, você pode especificar o tamanho de qualquer cabeçalho com o style atributo, usando a font-size propriedade CSS:

        
            
EXEMPLO:

        
<h1 style="font-size:60px;">Heading 1</h1>
        

        

        
 Parágrafos

        
O elemento HTML <p>define um parágrafo.

        
Um parágrafo sempre começa em uma nova linha e os navegadores adicionam automaticamente algum espaço em branco (uma margem) antes e depois de um parágrafo.

        
            
EXEMPLO:

        
<p>This is a paragraph.</p>

       
 <p>This is another paragraph.</p>

        

        

        
Exibição HTML

        
Você não pode ter certeza de como o HTML será exibido.


            
Telas grandes ou pequenas e janelas redimensionadas criarão resultados diferentes.

            
Com HTML, você não pode alterar a exibição adicionando espaços extras ou linhas extras em seu código HTML.

            
O navegador removerá automaticamente todos os espaços e linhas extras quando a página for exibida:

            
                
EXEMPLO:

            
<p>
                This paragraph
                contains a lot of lines
                in the source code,
                but the browser
                ignores it.
            </p>
                
            <p>
                This paragraph
                contains         a lot of spaces
                in the source         code,
                but the        browser
                ignores it.
            </p>
            

            

        
Regras Horizontais HTML

                
A <hr>tag define uma quebra temática em uma página HTML e geralmente é exibida como uma régua horizontal.

                
O <hr>elemento é usado para separar o conteúdo (ou definir uma alteração) em uma página HTML:

                
                    
EXEMPLO:

                
<h1>This is heading 1</h1>
                    <p>This is some text.</p>
                    <hr>
                    <h2>This is heading 2</h2>
                    <p>This is some other text.</p>
                    <hr>
                

                

                
A <hr>tag é uma tag vazia, o que significa que não tem tag final.

        
Quebras de Linha HTML

            
O elemento HTML <br> define uma quebra de linha.

            
Use <br> se quiser uma quebra de linha (uma nova linha) sem iniciar um novo parágrafo:

            
            
EXEMPLO:

            
<p> This is<br> a paragraph<br> with line breaks.

             

            

             
A <br>tag é uma tag vazia, o que significa que não tem tag final.

        
Elemento <pre> em HTML

            
O elemento HTML <pre> define o texto pré-formatado.

            
O texto dentro de um <pre> elemento é exibido em uma fonte de largura fixa (geralmente Courier) e preserva espaços e quebras de linha:

        
            
EXEMPLO:

        
<pre>
            My Bonnie lies over the ocean.
          
            My Bonnie lies over the sea.
          
            My Bonnie lies over the ocean.
          
            Oh, bring back my Bonnie to me.
        </pre>
        

        

        
 Estilo HTML

            
O atributo HTML styleé usado para adicionar estilos a um elemento, como cor, fonte, tamanho e muito mais.

            
O atributo de estilo HTML

             
A definição do estilo de um elemento HTML pode ser feita com o style atributo.

             
O atributo HTML style tem a seguinte sintaxe:

             
                
EXEMPLO:

             
<tagname style="property:value;"></tagname>
             

                

             
A propriedade é uma propriedade CSS. O valor é um valor CSS.

            
Cor de fundo

             
A propriedade CSS background-color define a cor de fundo de um elemento HTML.

             
                
EXEMPLO:

             
Defina a cor de fundo de uma página como azul-claro:


               
 <body style="background-color:powderblue;">
                
                <h1>This is a heading</h1>
                <p>This is a paragraph.</p>
                
                </body>

                
Defina a cor de fundo para dois elementos diferentes:


                   
 <body>
                    
                    <h1 style="background-color:powderblue;">This is a heading</h1>
                    <p style="background-color:tomato;">This is a paragraph.</p>
                    
                    </body></p>
             

    

            
Cor do Texto

            
A propriedade CSS colordefine a cor do texto para um elemento HTML:

    
                
EXEMPLO:

                
<h1 style="color:blue;">This is a heading</h1>
                    <p style="color:red;">This is a paragraph.</p>
                

    

            
Fontes

                
A propriedade CSS font-family define a fonte a ser usada para um elemento HTML:

    
                
EXEMPLO:

                
<h1 style="font-family:verdana;">This is a heading</h1>
                    <p style="font-family:courier;">This is a paragraph.</p>
                

    

            
Tamanho do Texto

                
A propriedade CSS font-size define o tamanho do texto para um elemento HTML:

    
                    <
EXEMPLO:

                
<h1 style="font-size:300%;">This is a heading</h1>
                    <p style="font-size:160%;">This is a paragraph.</p>
                

    

            
Alinhamento do Texto

                
A propriedade CSS text-align define o alinhamento horizontal do texto para um elemento HTML:

    
                    <
EXEMPLO:

                
<h1 style="text-align:center;">Centered Heading</h1>
                    <p style="text-align:center;">Centered paragraph.</p>
                

    

            
Resumindo todos os estilos citados acima:

                    
    
                        
  • Use o style atributo para estilizar elementos HTML
    • 
                        
  • Use background-color para cor de fundo
    • 
                        
  • Use color para cores de texto
    • 
                        
  • Use font-family para fontes de texto
    • 
                        
  • Use font-size para tamanhos de texto
    • 
                        
  • Use text-align para alinhamento de texto

 Comentários HTML

                
Os comentários HTML não são exibidos no navegador, mas podem ajudar a documentar seu código-fonte HTML.

                
Você pode adicionar comentários à sua fonte HTML usando a seguinte sintaxe:

            
                    <
EXEMPLO:

                
 <!-- Aqui está o seu comentário --> 

    

                
Observe que o ponto de (!) só é utilizado no início

                
Os comentários podem ser usados ​​para ocultar o conteúdo.

                
Isso pode ser útil se você ocultar o conteúdo temporariamente

                
Você também pode ocultar mais de uma linha. Tudo entre o <!--e o --> ficará oculto na tela.

                
Comentários também são ótimos para depurar HTML, porque você pode comentar linhas de código HTML, uma de cada vez, para procurar erros.

                
                    <
EXEMPLO:

                
 <p>Este é um parágrafo</p>
                <!--
                <p>Aqui pode ser uma imagem de um lindo cuscuz:</p>
                <img border="0" src="pic_trulli.jpg" alt="Trulli">
                -->
                <p>Aqui também pode ser um parágrafo.</p>

Sintaxe de Imagens HTML

        
A tag HTML <img>é usada para incorporar uma imagem em uma página da web.

        
As imagens não são tecnicamente inseridas em uma página da web; as imagens estão vinculadas a páginas da web. A <img>tag cria um espaço de retenção para a imagem referenciada.
   
        
A <img>tag está vazia, contém apenas atributos e não possui uma tag de fechamento.

        
A <img>tag tem dois atributos obrigatórios: src - Especifica o caminho para a imagem, alt - Especifica um texto alternativo para a imagem 

    
            <
EXEMPLO:

        
 <img src="url" alt="alternatetext">

    
  
        
Resolução de imagens

        
A resolução das imagens está ligada a quantidade de pixels que você define.

        
            <
EXEMPLO:

       
 <p> <img src="images/Cuscuz com ovos.jpg" alt="" width="128" height="128"></p>
        
 

       
 <p><img src="images/Cuscuz com ovos.jpg" alt="" width="100" height="128"></p>
        

        
<img src="images/Cuscuz com ovos.jpg" alt=""style="width:128px;height:128px;"></p>

        

        
<img src="images/Cuscuz com ovos.jpg" alt=""style="width:100px;height:128px;"></p>

 Listas 

    
As listas HTML permitem que os desenvolvedores da Web agrupem um conjunto de itens relacionados em listas. Elas podem ser ordenadas ou não

    
Listas não ordenadas
     
    
Uma lista não ordenada começa com a <ul>tag. Cada item da lista começa com a <li>tag.
 
    
Os itens da lista serão marcados com marcadores (pequenos círculos pretos) por padrão:

    
        
EXEMPLO:

        

            
  • Cuscuz
    • 
            
  • Ovo
    • 
            
  • Milho
    • 
          
    
        
    
          
    Listas Ordenadas
    
          
    Uma lista ordenada começa com a <ol>tag. Cada item da lista começa com a <li>tag.
    
          
    Os itens da lista serão marcados com números por padrão:
    
          
                
    EXEMPLO:
    
            
    
                
      
                    
    1. Cuscuz
      2.
