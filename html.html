Atributos em HTML

Dentro do HTML, existem atributos que ajudam a controlar o comportamento do elemento. São palavras especiais dentro da tag de abertura que modificam um tipo de elemento.

Todos os elementos podem ter atributos, onde estes contribuem com informações adicionais sobre os elementos. São sempre especificados na tag de início e geralmente vêm de pares de nome como: name="value"

Atributo HREF

Nesta tag, podemos atribuir uma URL de uma página que é definida pela tag <a>. 

                
EXEMPLO:


Adição de URL à tag <a>
    
<a> href="https://www.github.com">Github</a>

Atributo SRC

Este atributo é usado para adicionar uma imagem na página, por meio da tag <img>.

SRC significa "source" (fonte). Logo, o valor que o atributo vai incorporar será a URL da imagem que você adicionou. Ela aponta para o local onde a imagem está armazenada. 

                
EXEMPLO:


Adição de imagem na tag <img>

<img> src="cuscuz-com-ovo.png">

Existem duas maneiras de especificar a URL usando <src>:

  1. URL Absoluta: Aqui você você adiciona URL de uma imagem de outro site
    2.                     
    EXEMPLO:
    
    
    <src="https://www.github.com/images/cuscuz-com-ovo.png">
  2. URL Relativo:Imagem hospedada no site, mas a URL não tem o nome do domínio. Se não tiver barra, será relativa à página atual.
    3.                         
    EXEMPLO:
    
        
    <src="img_girl.jpg">

    Com barra, será relativa ao domínio. E é muito bom usar dessa forma, pois não quebra a URL, caso você mude de domínio.

                        
    EXEMPLO:
    
    
    <src="/images/img_girl.jpg">

Atributos WIDTH e HEIGHT

Estes atributos servem para configurar a imagem da tag <img>, especificando largura e altura da imagem, respectivamente. E, adicionando os pixels (tamanho que executa na tela).

                
EXEMPLO:


Configurando tamanho da imagem

<img src="cuscuz-com-ovo.png" width="500" height="600">

Atributo ALT

Este atributo obrigatório para a tag <img> especifica um texto alternativo para uma imagem, caso ela não possa ser exibida por alguma razão.

                
EXEMPLO:


Adicionando alt
<img src="cuscuz-com-ovo.png" alt="Comendo um cuscuzinho com ovo">

Caso a imagem não exista, ou a URL esteja errada, irá aparecer o que está dentro do atributo.

Atributo STYLE

Usado para implementar estilos aos elementos: cor, fonte e vários outro.

                
EXEMPLO:


Adição de parágrafo
<style="color:rosa;">Parágrafo rosa

Atributo LANG

Precisa atribuir o lang dentro do <html>, para declarar o idioma da página. Veja um exemplo que especifica o idioma como português do Brasil:

                
EXEMPLO:


idioma português do Brasil 
    <!DOCTYPE html>
    <html lang="pt-br">
    <body>
    ...
    </body>
    </html>

Atributo do TÍTULO

Ajuda a definir informações extras sobre um elemento. Ele exibe uma mensagem acima do texto assim que você passa o mouse sobre o elemento.

                
EXEMPLO:


Adição do title
    < p title="Melhor ainda com um ovinho frito">Cuscuz é maravilhoso!</p>

Cuscuz é maravilhoso!

Citações

As citações são vistas em HTML para fazer referências ou citações a outros conteúdos de outras páginas, por exemplo. com outras peculiaridades, ajudando a manter uma boa semântica e eficiente para o HTML5.

Os elementos de citação são:

  1. <bloquote>
  2. <q>
  3. <abbr>
  4. <address>
  5. <cite>
  6. <bdo>

<bloquote>

Elemento usado para definir uma seção citada de uma fonte distinta à página Web.

                    
EXEMPLO:

    
Bloquote
        <p> Texto do site que introduz as vantagens de se comer cuscuz<p>
        <bloquote> cite="http://www.cuscusbest.com/index.html">
        ...
        </bloquote>

<q>

Elemento usado para criar citações curtas.

                    
EXEMPLO:

    
Citação curta
        <p>Comer cuscuz faz bem, pois como todo baiano diz <q>cuscuz é vida!
        </q></p>

<abbr>

Esta tag especifica uma abreviação, como "CSS", "Sr.", "BR".

Caso você coloque uma abreviação de título global, ao passar o mouse sobre o elemento, exibe uma caixa de mensagem com a descrição da abreviação.

                
EXEMPLO:

    
Abreviação 
        <p> A <abbr title="Associação dos Amantes de Cuscuz">ACC</abbr> foi criada na Bahia, em 1903. </p> rs

<address>

Esta tag especifica os dados do contato do autor ou dono de um documento como e-mail, endereço local, telefone, entre outros.

                    
EXEMPLO:

    
Endereço
    <address>
    Escrito por Gean Martins
    Entre no meu site: 
    Gean.com 
    BR
    <address>

<cite>

Usa-se essa tag para criar títulos de um documento criativo como música, livro, etc.

    
Título criativo

    <p><cite> Crime e Castigo</cite>, por Dostoiévski.</p>

<bdo>

O Bi-Directional Override é uma tag usada para substituir a direção de um texto.

        
Substituição da direção do texto
        <bdo> dir = "rtl"> O cuscuz é feito com milho e muito amor </bdo>

    Icon em HTML

    O Icon ou favicon é uma imagem pequena ao lado do título da página no browser. Para adicionar uma favicon, você precisa usar uma imagem pequena, já que o título tem um espaço muito pequeno.

    Ela é exibida do lado do título.

    Para adicionar a favicon, implemente um <link> ao seu arquivo "index.html" depois do <title>:

            
    Adicionando icon à página
        
        <!DOCTYPE html>
        <html>
        <head>
            <title> Aprecie o Cuscuz </title>
            <link rel="icon" type="image/x-icon" href="/images/favicon-cuscuz.png">
        </head>
        <body>
        <h1> CUSCUZ QUENTINHO </h1>
        <p> Tem coisa melhor que cuscuz? Não teeem. </p>
        </body>
        </html>

    Salve o arquivo "index.html", atualize sua página no seu browser (tente F5). Você verá o icon ao lado do título da página.

    Classes HTML

    Este atributo é usado para determinar uma classe num elemento HTML. E inúmeros elementos podem compartilhar a mesma classe.

    A tag <class> geralmente é usada para indicar um nome de classe numa folha de estilo.

    No exemplo abaixo, veja que têm três <div> com uma classe "ingredientes". Todos estarão com o estilo .ingredients, dentro do <head>. 

            
     Adicionando classe
            
        <!DOCTYPE html>
        <html>
        <head>
        <style>
        .ingredients {
            background-color: yellow;
            color: white;
            border: 2px solid black;
            margin: 20px;
            padding: 20px;
        }
        </style>
        </head>
        <body>

        <div class="ingredientes">
        <h2> Farinha de Milho </h2>
        <p> A farinha de milho é o principal ingerdiente do cuscuz</p>
        </div>

        <div class="ingredientes">
        <h2> Sal ou Açúcar </h2>
        <p>Adiciona sal ou açúcar à farinha</p>
        </div>

        <div class="ingredientes">
        <h2> Água </h2>
        <p> Adicione água em tudo e mexa bem</p>
        </div>

        </body>
        </html>

    Várias classes

    Pode haver um elemento com mais de uma classe. Separam-se o nome das classes, como no exemplo:

            
     Adicionando várias classes

                    <h2 class="city main">Farinha de Milho</h2>
                    <h2 class="city">Açúcar ou Sal</h2>
                    <h2 class="city">Água</h2>
