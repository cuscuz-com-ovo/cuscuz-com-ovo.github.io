HTML

O QUE É?

A linguagem de Marcação de Hipertexto (HTML) é uma linguagem que está inserida na maioria das páginas da internet e APPs online, sinalizando aos servidores da web como se deve tratar a estrutura de um documento. 

                
EXEMPLO:


<!DOCTYPE html>
<html> 
<!-- Este é um comentário --><head>
<title> Títuto da minha página</title> 
</head> 
<body> 
Conteúdo da minha página em HTML!
</body> 
</html>
APRENDA HTML!

CSS

O QUE É?

O Cascading Style Sheet é uma linguagem utilizada para dar estilo aos elementos escritos em uma linguagem de marcação como o HTML, separando o conteúdo da representação visual do site. 

                
EXEMPLO:



body {
    background-color: blue;

}

p {
    color: black;
    padding: 10px;
    background-color: red;
    font-family: 'Barlow Condensed',sans-serif;
}
APRENDA CSS!
